## 🟢 第一套卷：科技与创新

| 题目关键词 | 步骤 | 内容要点 (背诵/默写) |
| :--- | :--- | :--- |
| **科技价值取向**<br>*(向善惠民)* | **第一步：抛出** | 任何成功的实践都是**真理尺度**和**价值尺度**的统一。 |
| | **第二步：解释** | **真理尺度**是指实践中人们必须遵循真理的指导。**价值尺度**是指在实践中人们按照自己的尺度和需要去认识世界和改造世界。 |
| | **第三步：关系** | **真理尺度**和**价值尺度**是**辩证统一**的关系。一方面，**价值尺度**必须以**真理**为前提。另一方面，人类自身需要的内在尺度，推动着人们不断发现新的**真理**。 |
| | **第五步：材料** | 只有推动科技创新向善惠民，才能符合**价值尺度**，从而达到**真理尺度**和**价值尺度**的统一。 |
| **科技双刃剑** | **第一步：抛出** | 科学技术是推动社会文明进步的重要力量。但是，科学技术的发展是一把**“双刃剑”**。 |
| | **第二步：解释** | 科学技术在促进经济和社会发展造福于人类的同时，也可能在一定条件下对人类的生存和发展带来消极后果。科学技术的作用会受到**主客观条件**的制约。 |
| | **第四步：方法** | 我们必须要用合理的**社会制度**保障科学技术的正确运用，使科学技术为人类造福。 |
| **宽容失败**<br>*(真理与谬误)* | **第一步：抛出** | **真理**与**谬误**、**成功**与**失败**是**对立统一**的。 |
| | **第二步：解释** | **真理**是人们对客观事物及其规律的正确反映；**谬误**是人们对客观事物及其规律的歪曲反映。 |
| | **第三步：关系** | 在确定的对象 and 范围内，**真理**和**谬误**是对立的，但是在一定条件下，它们又可以**相互转化**。在**批判谬误**中发展**真理**，是**谬误**向**真理**转化的一种形式。 |
| **创新曲折性**<br>*(否定之否定)* | **第一步：抛出** | **否定之否定规律**揭示了事物发展的**前进性**与**曲折性**的统一。 |
| | **第二步：解释** | **前进性**体现在每一次否定都会把事物发展推向一个新阶段；**曲折性**体现在事物发展过程中有暂时的停顿甚至倒退。事物的发展是一个**螺旋式上升、波浪式前进**的过程。 |

## 🔵 第二套卷：矛盾分析法

| 题目关键词 | 步骤 | 内容要点 (背诵/默写) |
| :--- | :--- | :--- |
| **学习借鉴**<br>*(非简单复制)* | **第一步：抛出** | 矛盾既具有**普遍性**，又具有**特殊性**，是二者的**辩证统一**。 |
| | **第二步：解释** | **矛盾的普遍性**，即矛盾无处不在、无时不有。**矛盾的特殊性**，即各个具体事物的矛盾、每一个矛盾的各个方面在发展的不同阶段上各有其特点。矛盾的特殊性决定了事物的不同性质。 |
| | **第三步：关系** | **矛盾的普遍性**即矛盾的**共性**，**矛盾特殊性**即矛盾的**个性**。任何事物的矛盾都是**共性**与**个性**的统一，**共性**寓于**个性**之中，**个性**也离不开**共性**。 |
| | **第四步：方法** | 既要看到事物矛盾的**普遍性**，也要善于分析矛盾的**特殊性**，做到**具体问题具体分析**。只有具体分析矛盾的特殊性，才能认清事物的本质和发展规律。 |
| | **第五步：材料** | 在学习借鉴其他城市经验的同时也要充分考虑各地、各产业的实际情况和特殊性，做到**因地制宜**。 |
| **共赢取代内卷**<br>*(辩证思维)* | **第一步：抛出** | **矛盾的同一性**和**斗争性**是矛盾的两种基本属性。 |
| | **第二步：解释** | **同一性**是指矛盾双方相互依存、相互贯通的性质和趋势；**斗争性**是指矛盾双方相互排斥、相互分离的性质和趋势。 |
| | **第三步：关系** | **矛盾的同一性**与**斗争性**是**辩证统一**的关系，**斗争性**寓于**同一性**之中，**同一性**通过**斗争性**体现出来。 |
| | **第四步：方法** | 事物的发展不仅表现为“**相反相成**”，而且表现为“**相辅相成**”，要求在对立中把握同一，在同一中把握对立。 |

## 🟠 第三套卷：联系与自然

| 题目关键词 | 步骤 | 内容要点 (背诵/默写) |
| :--- | :--- | :--- |
| **“借牧”模式**<br>*(普遍联系)* | **第一步：抛出** | 事物是**普遍联系**的，**联系**的观点是唯物辩证法的总观点和总特征之一。 |
| | **第二步：解释** | 联系具有**客观性**、**普遍性**、**条件性**。在尊重客观规律的前提下，人可以**改变和创造条件**，推动事物向着有利的方向发展。 |
| | **第四步：方法** | 要求人们善于分析事物的具体联系，从**动态**中考察事物的**普遍联系**。 |
| **人与自然**<br>*(生态文明)* | **第一步：抛出** | **自然地理环境**是人类社会生存和发展永恒的、必要的条件，是人们生活生产的自然基础。 |
| | **第二步：解释** | 合理利用自然资源，保护生态环境，是社会得以正常发展的必要条件。全球危机问题实际上也是**人与自然关系**的严重失衡。 |
| | **第四步：方法** | 人类必须**尊重自然、顺应自然、保护自然**。 |

## 🟣 第四套卷：认识论与思维

| 题目关键词 | 步骤 | 内容要点 (背诵/默写) |
| :--- | :--- | :--- |
| **不唯书只唯实**<br>*(物质统一性)* | **第一步：抛出** | 世界的**物质统一性原理**是辩证唯物主义最基本、最核心的观点。 |
| | **第二步：解释** | 世界是统一的，世界的本原只有一个，世界统一于**物质**。 |
| | **第四步：方法** | 必须做到**一切从实际出发**。即从客观事物存在和发展的规律出发，在实践中按照客观规律办事。 |
| | **第五步：材料** | “不唯上、不唯书、只唯实”就是反对盲从权威，反对**本本主义**，坚持**一切从实际出发**。 |
| **实践决定认识** | **第一步：抛出** | **实践决定认识**，**实践**是认识的来源、动力、目的和检验认识真理性的**唯一标准**。 |
| | **第二步：解释** | **实践**具有的**直接现实性**，是实践能够成为检验真理唯一标准的主要依据。 |
| **真理绝对/相对** | **第一步：抛出** | 任何**真理**都是**绝对性**与**相对性**的统一。 |
| | **第二步：解释** | **真理的绝对性**指主客观统一的确定性与发展的无限性；**真理的相对性**指对客观事物认识的有限性、不完善性。 |
| | **第三步：关系** | 二者相互包含，**绝对**之中有**相对**，**相对**之中有**绝对**；真理处于从相对性走向绝对性的转化过程中。 |
| | **第四步：方法** | 必须以**科学态度**对待一切真理，既要反对**绝对主义**（教条主义），又要反对**相对主义**。 |
| **地瓜经济**<br>*(思维能力)* | **解释考点** | 1. **辩证思维**：**联系**地、**发展**地、**全面**地、**系统**地观察事物。<br>2. **系统思维**：把握好**全局与局部**、**当前与长远**的关系。<br>3. **战略思维**：强调思维的**整体性、全局性、长期性**，是高瞻远瞩、统揽全局的能力。 |
